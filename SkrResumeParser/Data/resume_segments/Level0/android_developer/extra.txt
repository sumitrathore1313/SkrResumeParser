I created CheaterCheater as a simple, clean, and easy to use multipurpose word game helper application for
Android.

CheaterCheater helps find all word combinations word game hands including on board pieces, solve Scrabble,
Words With Friends and many other popular word games with this easy to use app! CheaterCheater also
contains a dictionary look up, so if you're playing a game of Scrabble and someone else plays a word, you
can immediately check that word on the current dictionary!

CheaterCheater  uses  the  open  source  ENABLE  word  list  to  give  you  the  most  viable  word  game  words
possible, and unlike many other word game helpers, CheaterCheater does this without any internet connection!
